# **App Name**: The Skill Garden

## Core Features:

- Garden Visualization: Render the user's skill garden with interactive stems and leaves. Leaf 'blooming' is reflected in color changes or subtle animation.
- Skill Planting: Allow users to add new skills ('seeds') and organize them into stems and leaves within their garden.
- Progress Tracking: Enable users to mark leaves as 'bloomed' and track their overall progress in the garden.
- Content Nurturing: Provide the ability to add notes, links, and reflections to each leaf, enriching their understanding.
- Data Persistence: Store user data, garden structure, and content in Firestore, ensuring real-time synchronization across devices.
- Garden Suggestion Tool: AI tool provides suggestions for related skills or resources based on the user's current garden.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) to evoke a sense of calm and growth.
- Background color: Very light desaturated lavender (#F5F5FF). Provides a calm and uncluttered backdrop.
- Accent color: Muted rose (#D8BFD8) to highlight interactive elements and bloomed leaves.
- Body font: 'Alegreya', a serif font to communicate an elegant, intellectual, contemporary feel.
- Headline font: 'Belleza', a sans-serif font well-aligned to design; it can also work for short paragraphs; otherwise it is paired well with Alegreya
- Use nature-inspired icons, such as leaves, stems, and flowers, to represent skills and learning stages.
- Employ a minimalist layout with generous whitespace to create a sense of calm and focus.
- Incorporate subtle animations, such as gentle leaf swaying or color transitions, to provide visual feedback without distraction.
